# Haskell Examples

From the book:

https://livebook.manning.com/book/get-programming-with-haskell/

How to run the modules:

```
> stack ghci
> :l 02-simple-functions.hs 
*Main> cat "Waleed" "Tim"
"Waleed + Tim <3"
```

Remember to load the file again to compile changes:
```:l 02-simple-functions.hs ```

Exit ghci: 
```ctrl-d```
